# Marka varlıkları — Myola BioSense

Bu klasör, sunum ve geliştirme için **marka tutarlılığı** referansıdır.

## Renkler (CSS tokenları ile uyumlu)

| Token | Hex | Kullanım |
|--------|-----|----------|
| Arka plan | `#050510` | Ana zemin |
| Mint | `#00f5d4` | CTA, vurgu, DFD akışları |
| Mor | `#8b5cf6` | İkincil vurgu, süreç kutuları |
| Altın | `#f59e0b` | Veri deposu çerçeveleri |
| Metin | `#e8e9ef` | Gövde |
| Soluk | `#9aa0b4` | Alt metin |

## Tipografi

- Başlıklar: **Space Grotesk**
- Gövde: **Inter**

## Logo

Gerçek logo dosyası yoksa: SVG veya PNG ekleyin ve `index.html` içinde hero veya footer’a bağlayın. Dosya adı önerisi: `brand-assets/logo.svg`.

## Not

Tasarım güncellemesi istendiğinde önce bu dosyadaki hex değerleri güncelleyin, ardından `style.css` içindeki `:root` ile eşleştirin.
